# Superpowers 参考

## 子技能速查

| 子技能 | 触发场景 | 输出 |
|:-------|:---------|:-----|
| `brainstorming` | 新项目/新功能开始时 | 需求文档 |
| `writing-plans` | 需求明确后 | 执行计划 |
| `executing-plans` | 开始编码 | 完成的代码 |
| `subagent-driven-development` | 任务可并行 | 并行完成的代码 |
| `test-driven-development` | 写代码时 | 通过的测试 |
| `requesting-code-review` | 代码写完 | 审查报告 |
| `receiving-code-review` | 收到审查意见 | 修改后的代码 |
| `finishing-a-development-branch` | 功能完成 | 清理后的分支 |

## 安装来源
- pip: `pip install agent-superpowers`
- 版本: 5.0.6
- 开源地址: https://github.com/obra/agent-superpowers
