# Superpowers 完整流程示例

## 场景：给个人主页添加一个博客功能

### 1. 头脑风暴
- **平台**：Web 端
- **功能**：写文章、展示文章列表、文章详情页
- **技术栈**：HTML + CSS + JS，无需后端，用 Markdown 文件存储
- **输出**：需求文档 `blog-requirements.md`

### 2. 写计划
- 任务1：创建博客列表页 `blog/index.html`
- 任务2：创建文章模板 `blog/template.html`
- 任务3：写 3 篇示例文章 `blog/posts/`
- 任务4：添加导航链接

### 3. TDD 开发
- 先写测试：文章列表是否正确渲染、链接是否正确
- 再实现功能
- 验证测试通过

### 4. 代码审查
- 检查 HTML 语义化
- 检查链接是否正确
- 检查移动端适配
