# Superpowers 工程化开发框架

一套完整的 AI 编程流程：头脑风暴 → 写计划 → TDD开发 → 代码审查 → 完成。

## 子技能

放在 `skills/` 目录下，按流程顺序：

| 阶段 | 技能 | 说明 |
|:----|:-----|:------|
| 💡 需求 | `brainstorming` | 苏格拉底式追问，理清需求 |
| 📋 计划 | `writing-plans` | 拆解为可执行任务 |
| 🔨 执行 | `executing-plans` | 按计划逐步执行 |
| 🤖 并行 | `subagent-driven-development` | 多子AI并行开发 |
| ✅ 测试 | `test-driven-development` | 测试驱动开发 |
| 👀 审查 | `requesting-code-review` | 审查代码 |
| 📥 接收 | `receiving-code-review` | 处理审查反馈 |
| 🏁 完成 | `finishing-a-development-branch` | 收尾清理 |

## 快速开始

当你说"开始一个新项目"时，我会先启动 `brainstorming`，一步步来。
